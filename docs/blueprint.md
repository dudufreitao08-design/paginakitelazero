# **App Name**: Kit Tela Zero Landing Page

## Core Features:

- Comprehensive Landing Page UI: Develop all specified sections including Hero, Pain Points, Product Offer, Bonuses, Testimonials, Pricing, Guarantee, FAQ, Final CTA, and Footer with structured content and layout as per user requirements.
- Adaptive Responsive Design: Implement a mobile-first responsive strategy targeting a 375px base for mobile, with breakpoints to optimize display for tablets and desktop layouts (e.g., column changes for grids and cards).
- Interactive FAQ Section: Implement an accordion functionality for the FAQ section using native JavaScript for a smooth user experience.
- Dynamic Mobile Sticky CTA: Develop a JavaScript-controlled sticky call-to-action bar that appears at the bottom of the screen on mobile devices after the hero section is scrolled past, and disappears when the pricing section is visible.
- AI-Powered CSS Product Mockup Generator Tool: Integrate a generative AI tool that can create CSS-only visual mockups of the 'Kit Tela Zero' product, dynamically adapting its components ('em casa', 'no carro', 'na rua', 'bônus') based on configuration parameters provided, ensuring consistency with the 'no external images' design constraint.
- Secure Payment Gateway Integration (Placeholder): Set up anchor links and placeholder functionality for Call-to-Action buttons that would connect to an external payment processing system for both the Basic and Premium plans.
- Basic Conversion Analytics: Implement basic client-side event tracking (e.g., Google Analytics or similar) on all CTA buttons and key interactions to monitor user engagement and conversion rates.

## Style Guidelines:

- The main accent and action color will be the vibrant orange '#FF6B35', conveying urgency, energy, and action for CTA buttons, badges, and focal elements as requested. The primary background will be pure white '#FFFFFF', maintaining a clean and direct aesthetic. A secondary accent color, a deep red-pink '#BA2340', will be used sparingly for subtle contrasts or secondary highlights, maintaining an analogous relationship with the main orange for design harmony.
- The typeface for all headlines and body text will be 'Poppins' (sans-serif), chosen for its contemporary, precise, and clean appearance, aligning with the direct and objective style requested. Note: currently only Google Fonts are supported.
- All visual icons, such as those for the benefits grid and pain point emojis, will be implemented using CSS-only shapes or inline SVGs to adhere to the 'no external images' constraint, ensuring a cohesive and performant visual style.
- A strict, mobile-first responsive layout will be prioritized (base 375px), adapting grid structures (e.g., 2 columns mobile, 3 desktop for benefits; 1 column mobile, 2/3 desktop for bonuses/testimonials) and section stacking (pricing cards) to ensure optimal viewing and navigation across all devices while strictly adhering to the specified section order.
- Smooth scrolling will be applied to all internal anchor links (e.g., CTA buttons linking to the pricing section). Interactive elements such as FAQ accordions will feature subtle expansion/collapse transitions, and the mobile sticky CTA bar will dynamically appear/disappear with smooth opacity and translate transformations based on scroll position.
